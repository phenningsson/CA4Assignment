How was this website created?

This website was created with a clean and straightforward design in mind. The usage of the nature-background sets the scene for the website and is centered around it. Using a color picker from the background image, a suitable color was selected for the footer and navigation bar to make them stand out while still fitting in nicely with the background. The white text with the arial, verdana, font gives the user a sense of familiarity with familiar colors and a standard font the user has most likely encountered before. The website is built using HTML, CSS, JavaScript as well with node.js, socket.io & express with server-client communication enabled through socket.io. Want to know more? Read the submission report at README.md.
